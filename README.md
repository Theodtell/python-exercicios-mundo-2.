# Python Exercises – Mundo 2

This repository contains Python exercises completed during **Mundo 2** of the Python course by Gustavo Guanabara (Curso em Vídeo).

The goal of these exercises is to strengthen programming logic and deepen the understanding of control structures in Python.

---

## Topics Practiced

During this stage of the course, the following concepts are practiced:

- Conditional structures
- Nested conditionals
- `for` loops
- `while` loops
- Counters and accumulators
- Basic program logic

---

## Structure

The exercises follow the numbering used in the course.

Example:
ex036.py
ex037.py
ex038.py


Each file represents a small program designed to practice a specific programming concept.
